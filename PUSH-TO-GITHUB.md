# How to Push to GitHub

## Step 1: Open Command Prompt or PowerShell in this folder
- Right-click inside this folder
- Select "Open in Terminal" or "Open PowerShell window here"

## Step 2: Run these commands one by one:

```bash
git init
git add .
git commit -m "Initial commit: Bay Area Direct homepage"
git branch -M main
git remote add origin https://github.com/bayareadirect/bayareadirect-bayareadirect-website.git
git push -u origin main
```

## That's it!

Your code will be pushed to GitHub and you can connect it in Claude Code.

## To connect in Claude Code:
1. Open Claude Code (from the dropdown)
2. Select "Open Repository"
3. Choose: bayareadirect/bayareadirect-bayareadirect-website
4. Start building!
