# Bay Area Direct Website

Same-day courier service website for the San Francisco Bay Area.

## About
Bay Area Direct provides reliable same-day delivery services throughout the Bay Area, serving businesses and individuals with urgent delivery needs.

## Features
- Responsive design
- Modern UI with Apple-inspired aesthetics
- Service listings and coverage areas
- Contact form for pickup requests
- Mobile-friendly navigation

## Tech Stack
- HTML5
- CSS3 (custom styling)
- Vanilla JavaScript
- Google Fonts (SF Pro Display, Inter)
